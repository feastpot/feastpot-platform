STEP 1 — TopNav: back button on all inner pages
File: apps/web/src/components/layout/top-nav.tsx
═══════════════════════════════════════════════════════

The TopNav currently shows a page title on inner pages with no navigation control.
Add a back arrow to the left of the title on all non-homepage routes.

import { useRouter, usePathname } from 'next/navigation'
import { ChevronLeft } from 'lucide-react'

// Determine if we are on an inner page
const isHome = pathname === '/'
const isVendorProfile = pathname.startsWith('/vendors/') && pathname !== '/vendors'
const isCheckout = pathname === '/checkout'
const isOrderTracking = pathname.startsWith('/orders/')

// Back destination logic
const getBackPath = (): string => {
  if (isVendorProfile) return '/vendors'
  if (isCheckout) return router.back() as any  // go back to vendor page
  if (pathname.startsWith('/orders/') && pathname.endsWith('/tracking')) return '/account/orders'
  if (pathname.startsWith('/orders/') && pathname.endsWith('/review')) return '/account/orders'
  if (pathname.startsWith('/account/')) return '/account'
  if (pathname.startsWith('/legal/')) return '/'
  return '/'
}

On all non-homepage routes, replace the title-only left section with:

<div className="flex items-center gap-1">
  <button
    onClick={() => {
      const back = getBackPath()
      if (typeof back === 'string') router.push(back)
      else router.back()
    }}
    className="flex items-center justify-center w-9 h-9 rounded-full hover:bg-cream transition-colors touch-target -ml-1"
    aria-label="Go back"
  >
    <ChevronLeft className="w-6 h-6" style={{ color: '#1C1C1A' }} strokeWidth={2.5} />
  </button>
  <h1 className="text-[17px] font-semibold" style={{ color: '#1C1C1A' }}>{title}</h1>
</div>

STEP 3 — Scroll position preservation on back navigation
File: apps/web/src/app/vendors/page.tsx
═══════════════════════════════════════════════════════

When a user taps a vendor card and then presses back, the /vendors list scrolls
back to the top. This is disorienting — the user expects to return to where they were.

Add scroll position persistence:

'use client'
import { useEffect, useRef } from 'react'

// Store scroll position before navigating away
useEffect(() => {
  const handleBeforeUnload = () => {
    sessionStorage.setItem('vendors-scroll', window.scrollY.toString())
  }
  window.addEventListener('beforeunload', handleBeforeUnload)
  return () => window.removeEventListener('beforeunload', handleBeforeUnload)
}, [])

// Restore scroll position on mount
useEffect(() => {
  const saved = sessionStorage.getItem('vendors-scroll')
  if (saved) {
    window.scrollTo({ top: parseInt(saved), behavior: 'instant' })
    sessionStorage.removeItem('vendors-scroll')
  }
}, [])

Also use Next.js scroll={false} on vendor card links to prevent scroll-to-top:
<Link href={`/vendors/${vendor.slug}`} scroll={false}>
