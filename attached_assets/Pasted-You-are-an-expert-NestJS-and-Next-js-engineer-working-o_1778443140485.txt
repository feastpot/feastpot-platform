Create the following files:

1. addresses.module.ts
   - Imports PrismaModule
   - Provides AddressesService, AddressesRepository
   - Exports AddressesService (OrdersModule will import it)

2. addresses.controller.ts
   Routes:
   GET    /v1/addresses          — list all addresses for authenticated user
   POST   /v1/addresses          — create new address
   PATCH  /v1/addresses/:id      — update address (own only)
   DELETE /v1/addresses/:id      — delete address (own only, cannot delete if active order uses it)

   All routes require authentication (no @Public()).
   Use @CurrentUser() to get the authenticated user.

3. addresses.service.ts
   findAll(userId: string): return all addresses for user, default address first
   
   create(userId: string, dto: CreateAddressDto):
   - Validate UK postcode format using regex: /^[A-Z]{1,2}[0-9][0-9A-Z]?s?[0-9][A-Z]{2}$/i
   - Geocode postcode to lat/lng using free postcodes.io API:
     GET https://api.postcodes.io/postcodes/{postcode}
     Extract latitude and longitude from response.result
   - If isDefault is true, set all other addresses for this user to isDefault=false first
   - Create and return the new address
   
   update(id: string, userId: string, dto: UpdateAddressDto):
   - Verify address belongs to userId, throw NotFoundException if not
   - If isDefault is true, set all others to false first
   - Update and return
   
   delete(id: string, userId: string):
   - Verify ownership
   - Check no active orders (status IN [pending, accepted, preparing, dispatched]) use this deliveryAddressId
   - If in use, throw ConflictException: "This address is used by an active order and cannot be deleted"
   - Delete and return { deleted: true }
   
   setDefault(id: string, userId: string):
   - Set all user addresses isDefault=false
   - Set this address isDefault=true

4. addresses.repository.ts — all Prisma queries isolated here

5. DTOs:
   CreateAddressDto:
   - label?: string (max 60 chars)
   - line1: string (required, min 3)
   - line2?: string
   - city: string (required)
   - postcode: string (required, validated with UK regex)
   - isDefault?: boolean (default false)
   
   UpdateAddressDto: Partial<CreateAddressDto>

6. Register AddressesModule in app.module.ts imports array
