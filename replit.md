# Node.js App

Minimal Node.js + Express starter running on port 5000.

## Stack
- Node.js (ES modules)
- Express 4
- Static files served from `public/`

## Scripts
- `npm run dev` / `npm start` — start the server (`node server.js`)

## Replit setup
- Workflow: `Start application` runs `npm run dev` on port 5000 (webview)
- Deployment: autoscale, run command `node server.js`
